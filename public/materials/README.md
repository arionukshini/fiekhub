# Public Materials

This is the active static file location for the current GitHub Pages version of
FIEK Hub.

Files placed here are copied into the production build and can be opened by the
browser. For example:

- `materials/provime-pranuese/2019/example.pdf`
- `materials/fiek/viti1/example.pdf`

Long term, large academic files should move to Supabase Storage. Until then,
this folder is the source for static PDFs that need to be viewable in the app.
